# Public source adapters

## LipoBART

- Repository: <https://github.com/Sanofi-Public/LipoBART>
- Scope: ionizable-lipid SMILES와 transfection prediction benchmark
- 주의: upstream 라이선스는 학술 연구 및 비상업 이용 조건을 둔다.

## LNPDB

- Database: <https://lnpdb.molcube.com/>
- Paper: <https://doi.org/10.1038/s41467-026-68818-1>
- Scope: LNP composition, experiment, performance, simulation metadata

## RNA cargo size-gradient benchmark

- Paper: <https://doi.org/10.1038/s41598-024-52685-1>
- Scope: 10, 21, 96, 996, 1929 nt RNA cargo와 benchmark formulation 비교

이 저장소는 위 데이터의 레코드를 재배포하지 않는다. 사용자가 각 원출처의 조건에 동의하여
`data/external/`에 둔 파일을 canonical schema로 연결하는 방식으로 확장한다.
